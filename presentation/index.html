Request life cycle

So you validate your order on Amazon

Naive and simplified, yet accurate

You clicked "Confirm"

  • Your browser (Chrome, Firefox, IE, ...) is sending a request
  • This request contains a URL, some cookies informations
  • This request is hitting a server (some physical computer)

So there's this request

  • The request is targeting a precise port (just a number)
  • The server has a program listening (usually an Apache Server)
  • So yeah, there are two servers: a physical server and a software server
  • This last server is here to handle the request
  • Apache knows what PHP program to call

The program is running

  • The request is processed to identify the user and the order
  • The request is processed to confirm the order
  • Each of these operations implies a "complex" logic
  • Each of these operations implies a discussion with a database
  • In the end, a new HTML page is returned

array_max

                    function array_max($a) {
                        $max = NULL;
                        foreach ($a as $value) {
                            if (!isset($max) || ($value > $max)) {
                                $max = $value;
                            }
                        }
                        return $value;
                    }

                    // PHP native function: max($a);

array_max_index

                    function array_max_index($a) {
                        $maxIndex = NULL;
                        $max = NULL;
                        $length = count($a);
                        for ($i = 0; $i < $length; $i++) {
                            $value = $a[$i];
                            if (!isset($max) || ($value > $max)) {
                                $maxIndex = $i;
                                $max = $value;
                            }
                        }
                        return $maxIndex;
                    }

                    // PHP native function: array_search(max($a), $a);

is_sorted

                    function is_sorted($a) {
                        $previousValue = NULL;

                        foreach ($a as $currentValue) {
                            if (isset($previousValue) && ($previousValue > $currentValue)) {
                                return FALSE;
                            }
                            $previousValue = $currentValue;
                        }
                        return TRUE;
                    }

                    // PHP native function: none

array_sum

                    function our_array_sum($a) {
                        $sum = 0;

                        foreach ($a as $value) {
                            $sum += $value;
                        }

                        return $sum;
                    }

                    // PHP native function: array_sum

reverse

                    function reverse($a) {
                        $length = count($a);
                        $reversedArray = array();

                        for ($i = $length - 1; $i >= 0; $i--) {
                            $reversedArray[] = $a[$i];
                        }
                        return $reversedArray;
                    }

                    // PHP native function: array_reverse
